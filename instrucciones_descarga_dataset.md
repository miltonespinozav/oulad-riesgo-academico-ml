# Instrucciones para descargar el dataset OULAD

El notebook intenta descargar automáticamente el dataset desde UCI Machine Learning Repository.

Si la descarga automática falla:

1. Ingrese a la página oficial del dataset en UCI Machine Learning Repository.
2. Busque: **Open University Learning Analytics dataset**.
3. Descargue el archivo ZIP del dataset.
4. Suba el archivo ZIP a Google Colab.
5. Coloque el archivo como `/content/oulad_project/oulad.zip`.
6. Ejecute nuevamente las celdas de extracción y carga.

Archivos esperados dentro del dataset:

- courses.csv
- assessments.csv
- studentAssessment.csv
- studentInfo.csv
- studentRegistration.csv
- studentVle.csv
- vle.csv

No suba datos privados ni información sensible. Si usa datos propios, deben estar anonimizados.
